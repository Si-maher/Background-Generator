Background Generator review

The project

This project creates rgba colors and then changes the background of the body with the new selected colors. A gradient which flows to the right is also used.
Technologies used are; JavaScript, HTML and CSS. Javascript is used to dynamically change the body into the new colors.

How it works

Firstly, a number of variables using let are declared and are connected to the DOM through getElementbyId and byElement. On the index.html these variables are connected to various elements. This allows Javascript to know what to search and render.

A function is created that when called, will render the background color. It targets the new color's rgba value and is set as the CSS textContent.

Problems

The main problem was due to me having class instead of ID inside the html form. This returned a value in the console of 'cannot read of undefined'. After some careful code view, this was observed and duly corrected.
